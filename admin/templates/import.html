Format du fichier

Le fichier CSV doit utiliser le point-virgule (;) comme séparateur.

Colonnes disponibles :

idID numérique ou tcgdex_id (pour modification)
nameNom de la carte
local_idNuméro dans le set (ex: 25, 001, H01)
set_nameNom du set (pour affichage)
set_idID du set (pour création)
set_cardcount_officialTotal officiel du set (ex: H32)
variantNORMAL, REVERSE, HOLO, FIRST_ED
card_number_formatLOCAL_ONLY, LOCAL_TOTAL, PROMO
card_number_paddedtrue/false - Padding zeros
ebay_queryRequête eBay personnalisée

Mode modification (id renseigné) :

id;name;local_id;set_name;set_cardcount_official;card_number_format;card_number_padded;ebay_query

svp-001-NORMAL;;;;;PROMO;true;
